# :checkered_flag: FixNow

O FixNow é uma aplicaçõa web que facilita o agendamento de serviços de manuntenção e reparo, conectando clientes a prestadores de serviço (como eletricistas, encadores, etc.). O Sistema visa agilizar o processo de contratação, garantindo transparência e praticidade sobre atendimentos.

## :bulb: Objetivo Geral
Desenvolver um sistema web completo de agendamento e gerenciamento de serviços técnicos, que possibilite a interação entre clientes e prestadores, oferecendo uma plataforma moderna, segura e eficiente para o controle de solicitações e atendimentos.

## :eyes: Público-Alvo
Clientes e prestadores de serviço

## :star2: Impacto Esperado
Otimizar processos de solicitação e prestação de serviço, melhorando a comunicação entre clientes-prestadores, além de promover visibilidade para profissionais autônomos 

## :people_holding_hands: Papéis ou tipos de usuário da aplicação

- Usuário não logado - Pode visualizar informações básicas sobre serviços disponíveis, mas não pode agendar pelo sistema;
- Cliente - Pode criar e gerenciar agendamentos, visualizar status, cancelar solicitações e editar perfil;
- Técnico (Prestador de serviço) - Pode visualizar agendamentos recebidos, confirmar ou marcar como concluido e editar disponibilidade;

## :triangular_flag_on_post:	 Principais funcionalidades da aplicação

Funcionalidades abertas: Visualização de serviços disponiveis (lista pública); Página inicial; Formulário de cadastro e login.
Funcionalidades restritas:
- Cliente: Criar e visualizar agendamentos; Cancelar solicitações; Editar perfil;
- Técnico: Genrenciar disponibilidade de horários; COnsultar solicitações; Atualizar status dos atendimentos (pendente, confirmado, concluido)

## :spiral_calendar: Entidades ou tabelas do sistema

Tabelas: Usuário, Serviço, Agendamento.
